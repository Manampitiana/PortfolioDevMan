## ğŸš€ Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone <repository-url>
cd Portfolio-man/frontend
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser and visit `http://localhost:5173`

## ğŸ¨ Customization

### Colors
The project uses a custom color scheme with gradients. You can modify the colors in:
- `src/index.css` - Global styles and custom CSS
- Individual components - TailwindCSS classes

- `src/pages/Home.jsx` - Skills, services, and hero content
